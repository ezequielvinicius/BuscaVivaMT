# Escopo do Projeto - BuscaVivaMT

## Dentro do Escopo ✅

### Features Principais
1. **Consulta Pública**
   - Lista paginada
   - Cards informativos
   - Status visual

2. **Sistema de Busca**
   - Campo de texto
   - Filtros múltiplos
   - Busca em tempo real

3. **Detalhamento**
   - Página individual
   - Todas informações
   - Galeria de fotos

4. **Colaboração Cidadã**
   - Formulário de avistamento
   - Upload de fotos
   - Marcação no mapa

### Técnico
- SPA com React
- TypeScript
- Tailwind CSS
- Docker
- Testes unitários

## Fora do Escopo ❌

- Backend/API
- Autenticação/Login
- Painel administrativo
- App mobile nativo
- Notificações push
- Chat/Mensagens
- Integração com redes sociais
